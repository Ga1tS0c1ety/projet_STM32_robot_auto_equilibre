# Projet STM32 - Robot Auto-Équilibré

Ce dépôt contient mon projet personnel autour de la STM32 et de la robotique embarquée.

L'objectif final est de construire progressivement un robot auto-équilibré basé sur une carte STM32 Nucleo-F446RE.

## Organisation actuelle

- `documentation/` : carnet de bord, documents techniques, schémas, notes générales.
- `seance1/` : première séance de prise en main STM32.
- `seance2/`, `seance3/`, etc. seront ajoutées progressivement.

## Objectifs du projet

- apprendre STM32CubeIDE ;
- comprendre les GPIO, UART, PWM, I2C, Timers ;
- utiliser une IMU MPU6050 ;
- piloter des moteurs avec un driver TB6612FNG ;
- lire des encodeurs ;
- implémenter un contrôle PID ;
- construire un robot auto-équilibré.
